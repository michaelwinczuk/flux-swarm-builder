# SwarmForge AI — Business Model

## Mission
Build the simplest, most powerful local-first swarm builder. Anyone can create a multi-agent team from markdown files and run it on their own hardware. Zero cloud dependency. Zero API cost for core operations.

## Product: FLUX Swarm Builder
- Open source CLI + visual canvas for building agent swarms
- Agents defined by .md cartridges — no code required to create an agent
- Knowledge system: JSON-LD graphs + BM25 search + algebraic reasoner
- Local model (Phi-3 3.8B) handles routing, scoring, expansion at zero cost
- API calls reserved for deep synthesis only (Opus for visionary thinking)
- Capability mesh: agents discover each other's abilities automatically

## Revenue Model
- Core tool: free and open source (MIT license)
- FLUX Pro: $4.99 one-time (advanced features, no subscription)
- Enterprise: custom swarm templates for specific industries
- Skills marketplace: community-built agent skills (open source with GitHub screener)

## Target Customers
- AI researchers who need multi-agent experimentation
- Small teams who want an "AI department" running locally
- Developers building agent-powered products
- Companies wanting private, on-premise AI workflows

## Competitive Advantages
1. Local-first: no data leaves your machine
2. File-based: the swarm IS a folder — zip it, git it, share it
3. Proven: built while competing in Parameter Golf (beat SOTA)
4. Cost: $0 for core operations, pennies for synthesis calls
