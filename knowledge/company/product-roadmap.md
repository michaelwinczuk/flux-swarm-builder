# SwarmForge AI — Product Roadmap

## Shipped (v0.1)
- CLI swarm runner (swarm.py) — mission in, report out, 10 seconds
- 6 specialist agents (causes, requires, solves, contradicts, tradeoff, systems)
- Capability mesh with typed subdirectories (llm/, tools/, viewers/, claws/)
- Capability token JSON schema for agent discovery
- Constitution-based governance
- Phi-3 3.8B local model for routing and scoring
- Knowledge loading from markdown files
- Audit trail (audit.json per agent)
- Benchmark scorecards (scorecard.json per agent)

## In Progress (v0.2)
- Visual canvas (React) for drag-and-drop agent creation
- Knowledge graph engine (JSON-LD + BM25 + algebraic reasoner from TTS)
- Library chaining — drop research folders, auto-indexed
- Real-time agent activity visualization
- Viewer oracles for live data feeds (price, weather, news)

## Planned (v0.3)
- Swarm templates (trading, research, customer service, content)
- Agent-to-agent communication via message bus
- Multi-model routing (local model for cheap tasks, API for hard tasks)
- Mobile companion app (FLUX — Your Digital Self)

## Future Vision
- Structured intelligence: skip tokenization, encode knowledge directly
- Agent personalities as distinct reasoning models
- BLE proximity social matching between user swarms
